Calculating Metrics

Availability = Bicycle Traffic Roads / Car Traffic Roads

Quality = (Total Cycleway Length / Bicycle Traffic Roads) * 100

berlin <- berlin |> mutate(
  availability = round(((total_bikelane_length + total_cycleway_length) / total_roads_length), 2), 
  quality = round((total_cycleway_length / (total_bikelane_length + total_cycleway_length)) * 100, 2)
)
amsterdam <- amsterdam |> mutate(
  availability = round(((total_bikelane_length + total_cycleway_length) / total_roads_length), 2), 
  quality = round((total_cycleway_length / (total_bikelane_length + total_cycleway_length)) * 100, 2)
)

This code is run so that tmap is compatible with the font I want to use.

windowsFonts(`Century Gothic` = windowsFont("Century Gothic"))

Availability

Bike Infrastructure Availability (per km Motor Vehicle).

Note: I used limits here because we dont want tmap here to scale the legends; I want the visualization to capture the contrast by having the same colors for the values. In this way it simulates uniform legend colors between Berlin and Amsterdam. I set the midpoint at 0.75 (proportion) so that the colors contrast a bit more for the lower-end (where most of Berlins values fall). This also has the added effect of making the high-end of proportion in Amsterdam be completely different from the 0.8 to 1.0 range.

ba <- tm_shape(berlin) + 
  tm_polygons(
    fill = "availability",
    fill.scale = tm_scale_continuous(values = "roma",
                          limits = c(0,1),
                          midpoint = .75),
    fill.legend = tm_legend(title = "Proportion (km per km)",
                            orientation = "landscape",
                            frame = FALSE)
  ) +
  tm_title_out("Berlin: Kilometers of Bike Paths per Motor Vehicle Roads",
              fontface = "bold",
              fontfamily = "Century Gothic") +
  tm_layout(
    legend.text.fontfamily = "Century Gothic",
    legend.title.fontfamily = "Century Gothic",
    text.fontfamily = "Century Gothic"  # optional, sets global fallback
  ) +
  tmap_options(component.autoscale = FALSE)
